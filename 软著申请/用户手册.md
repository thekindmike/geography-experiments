## 第一章 软件概述

### 1.1 软件简介
AIAI地理实验教学平台是一个基于Web的地理实验教学辅助工具，专为初高中地理教师和学生设计。本平台将传统地理实验数字化，提供完整的实验资源库和互动模拟功能，让地理实验更高效、更有趣。

### 1.2 主要功能
1. **实验资源库**：收录初高中地理实验18个，每个实验包含完整的实验指导
2. **互动模拟实验**：18个实验全部配备互动模拟器，学生可在浏览器中直接操作
3. **智能搜索与筛选**：支持多维度搜索和分类筛选
4. **教师演示模式**：专为课堂教学设计，大字体、高对比度
5. **响应式设计**：适配手机、平板、电脑等多种设备

### 1.3 运行环境
- **操作系统**：Windows, macOS, Linux, iOS, Android等
- **浏览器**：Chrome 80+, Firefox 75+, Safari 13+, Edge 80+等现代浏览器
- **网络**：无需联网，本地运行（如需在线访问，需联网）
- **硬件**：普通电脑或移动设备即可

## 第三章 功能详解

### 3.1 首页
**功能**：平台入口，展示平台特色和推荐实验

**操作**：
1. 打开平台，自动进入首页
2. 查看平台特色功能（完整实验库、互动模拟、教师演示模式等）
3. 点击"开始学习初中实验"或"开始学习高中实验"按钮，进入相应实验列表
4. 查看推荐实验，点击可查看详情

**界面说明**：
- **英雄区**：展示平台标题和简介
- **特色功能区**：展示6大特色功能
- **推荐实验区**：展示精选实验

### 3.2 初中实验列表
**功能**：展示所有初中地理实验，支持分类筛选

**操作**：
1. 点击顶部导航栏"初中实验"，或首页"开始学习初中实验"按钮
2. 查看初中实验列表（默认显示全部）
3. 点击分类筛选按钮（如"地球与地图"、"天气与气候"等），筛选相应主题的实验
4. 点击实验卡片，查看实验详情

**界面说明**：
- **筛选栏**：按主题分类筛选
- **实验卡片**：展示实验名称、难度、时长、标签等信息

### 3.3 高中实验列表
**功能**：展示所有高中地理实验，支持分类筛选

**操作**：
1. 点击顶部导航栏"高中实验"，或首页"开始学习高中实验"按钮
2. 查看高中实验列表（默认显示全部）
3. 点击分类筛选按钮（如"自然地理·大气"、"人文地理·城市"等），筛选相应主题的实验
4. 点击实验卡片，查看实验详情

### 3.4 实验详情页
**功能**：展示实验的完整信息，包括实验目的、原理、器材、步骤、数据记录、结果分析、思考题等

**操作**：
1. 在实验列表点击任一实验卡片
2. 查看实验详情，包括：
   - **实验信息**：名称、难度、时长、评分、标签
   - **实验目的**
   - **实验原理**
   - **实验器材**
   - **实验步骤**
   - **数据记录**（部分实验）
   - **结果分析**
   - **思考题**
3. 如该实验有互动模拟器，点击"开始实验"按钮，进入互动模拟

**界面说明**：
- **返回按钮**：点击返回实验列表
- **开始实验按钮**：如有互动模拟器，点击进入

### 3.5 互动实验页
**功能**：展示所有配备互动模拟器的实验

**操作**：
1. 点击顶部导航栏"互动实验"
2. 查看所有互动实验列表
3. 点击"开始实验"按钮，进入互动模拟器

### 3.6 互动模拟器
**功能**：在浏览器中直接操作，观察地理实验现象，加深理解

**操作**（以"经纬网定位实验"为例）：
1. 进入互动模拟器页面
2. 查看**控制变量说明**（了解实验的科学方法）
3. 在**自变量调节面板**调节参数（如纬度、经度）
4. 观察**因变量观察区**的实验结果（实时变化）
5. 点击"记录数据"按钮，将数据添加到**数据记录表**
6. 查看**科学结论分析框**，分析实验结论
7. 点击"重置"按钮，重置实验

**注意事项**：
- 每个互动模拟器的操作方式略有不同，请仔细阅读界面提示
- 建议先理解"控制变量说明"，再进行实验操作
- 实验数据会自动保存到浏览器本地存储（LocalStorage）

### 3.7 搜索功能
**功能**：快速查找所需实验

**操作**：
1. 点击顶部导航栏右侧的"搜索"图标（放大镜）
2. 在搜索框中输入关键词（如"经纬网"、"气温"等）
3. 实时显示搜索结果
4. 点击搜索结果，查看实验详情
5. 点击搜索框右侧"✕"按钮，关闭搜索栏

### 3.8 教师演示模式
**功能**：专为课堂教学设计，大字体、高对比度，方便教师课堂投影展示

**操作**：
1. 在任意页面，按快捷键 `F11` 进入全屏模式
2. 点击页面底部的"教师演示模式"按钮（如有）
3. 界面自动切换为大字体、高对比度模式
4. 按 `Esc` 键或点击"退出演示"按钮，退出演示模式

### 3.9 移动端使用
**功能**：在手机或平板上使用本平台

**操作**：
1. 在手机或平板浏览器中打开平台网址
2. 界面自动适配移动端
3. 点击左上角"菜单"图标，打开移动端菜单
4. 点击菜单项，切换页面

## 第四章 互动模拟器使用指南

### 4.1 控制变量法简介
本平台的互动模拟器采用"控制变量法"科学实验方法，即：
- **控制变量**：保持不变的条件
- **自变量**：主动改变的条件
- **因变量**：随自变量变化而变化的观察结果

### 4.2 互动模拟器通用操作
1. **阅读控制变量说明**：了解实验的科学方法
2. **调节自变量**：在自变量调节面板，使用滑块或按钮调节参数
3. **观察因变量**：在因变量观察区，实时观察实验结果
4. **记录数据**：点击"记录数据"按钮，将数据添加到数据记录表
5. **分析结论**：查看科学结论分析框，分析实验结论
6. **重置实验**：点击"重置"按钮，重置所有参数

### 4.3 典型互动模拟器说明

#### 4.3.1 经纬网定位实验（j001）
- **自变量**：纬度、经度
- **因变量**：经线、纬线、定位结果
- **操作**：调节纬度、经度滑块，观察经线纬线变化

#### 4.3.2 地球自转实验（j002）
- **自变量**：自转方向、转速
- **因变量**：昼夜交替、时差
- **操作**：点击"启动/暂停"按钮，观察昼夜交替

#### 4.3.3 等高线地形图绘制与判读（j004）
- **自变量**：地形类型、等高距
- **因变量**：等高线形态
- **操作**：选择地形类型，调节等高距，观察等高线变化

#### 4.3.4 气温测定与记录（j005）
- **自变量**：时间、季节、地点
- **因变量**：气温变化曲线
- **操作**：选择时间、季节、地点，点击"测量"，观察气温变化

## 第六章 常见问题解答

### 6.1 网站无法打开怎么办？
**答**：
1. 检查网络连接是否正常
2. 检查浏览器版本是否过旧（需Chrome 80+等现代浏览器）
3. 尝试清除浏览器缓存后重新打开
4. 如图案无法显示，检查是否拦截了JavaScript

### 6.2 互动模拟器无法操作怎么办？
**答**：
1. 检查浏览器是否支持JavaScript（需启用JavaScript）
2. 尝试刷新页面（Ctrl+F5强制刷新）
3. 如仍无法操作，尝试更换浏览器

### 6.3 实验数据无法保存怎么办？
**答**：
1. 检查浏览器是否允许本地存储（LocalStorage）
2. 如启用了"无痕模式"，浏览器可能不会保存数据
3. 建议关闭无痕模式，或使用普通模式浏览

### 6.4 手机上显示不正常怎么办？
**答**：
1. 尝试横屏显示（部分模拟器需横屏）
2. 尝试放大页面（双指放大）
3. 如使用移动端浏览器，建议下载桌面版浏览器（如Chrome桌面版）
